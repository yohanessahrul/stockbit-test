## Run Locally

Clone the project

```bash
  git clone https://github.com/yohanessahrul/stockbit-test.git
```

Go to the project directory

```bash
  cd stockbit-test
```

Install dependencies

```bash
  npm install sass
```

```bash
  npm install
```

Start the server

```bash
  npm start
```

  
## Tech Stack

**Client:** React, Redux, Sass
